PROJECTS

Museum Of Candy

A mock-up website inspired by the instagrammable Museum of Candy in New York. The purpose of this website is to demonstrate some skills in bootstrap 4 and the grid system, including responsiveness and mobile-friendly content.

HTML, CSS, and a little bit of javascript are used in this project.

You can view this project here: Museum Of Candy

Eco Mall

A group project completed during my undergrad using pure css and html to layout the website. The website is a mock online shopping website with special authenticating feature. Only users who are logged in can "shop" and review products.

HTML, CSS, javascript, ajax, jquery, SQL database, TinyMCE and php are used in this project.

You can view this project here: Eco Mall

Colour Guessing Game

A web application using HTML, CSS, and Javascript. The Colour Game application randomly selects an RGB color and the user must guess the colour. There are two levels, easy and hard.

You can view this project here: Colour Guessing Game
